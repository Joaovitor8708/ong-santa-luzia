# Abrigo Santa Luzia - Plataforma de Gestão Social 🐾

Este projeto foi desenvolvido como parte de um trabalho interdisciplinar (ADS - UNIBRA) com o objetivo de digitalizar os processos de doação e visibilidade da ONG Abrigo Santa Luzia. 

O sistema foi projetado para ser intuitivo e acessível, atendendo a voluntários com pouca familiaridade digital.

## 🛠️ Tecnologias e Ferramentas
* **Back-end:** PHP 8.3 e Framework Laravel 13
* **Banco de Dados:** MySQL
* **Front-end:** Blade Engine, HTML5, CSS3 e JavaScript
* **APIs & Integrações:** ViaCEP, Google Maps, V-Libras e Chatbot de suporte

## 🚀 Minha Contribuição
Neste projeto, atuei como **Desenvolvedor Back-end** e **QA (Quality Assurance)**, sendo responsável por:
* Estruturação da lógica de negócio e rotas utilizando Laravel.
* Modelagem do banco de dados MySQL.
* Implementação de integrações com APIs externas.
* Planejamento e execução de testes funcionais para garantir a estabilidade e usabilidade do sistema.
